Sesi 1: Perkenalan

Selamat datang di bagian pengenalan kursus. Di bagian ini, Anda akan mempelajari dasar-dasar pengembangan web dan mengenal konten kursus.

Pengembangan web melibatkan serangkaian keterampilan dan teknologi untuk membuat dan memelihara situs web atau aplikasi web. Berikut adalah beberapa dasar-dasar pengembangan web yang mungkin berguna untuk pemula: 1. HTML (Hypertext Markup Language): HTML adalah bahasa markah yang digunakan untuk membuat struktur dasar halaman web. Elemen HTML digunakan untuk menandai dan mendefinisikan konten, seperti teks, gambar, tautan, formulir, dan lainnya. 2. CSS (Cascading Style Sheets): CSS digunakan untuk mengatur tata letak, warna, dan gaya elemen-elemen HTML. Dengan CSS, dapat membuat tampilan yang konsisten dan menarik untuk situs web Anda. 3. JavaScript: JavaScript memberikan interaktivitas pada halaman web. Ini dapat digunakan untuk mengubah konten, merespons input pengguna, dan berkomunikasi dengan server untuk mengambil atau mengirim data. 4. HTTP dan HTTPS: Protokol HTTP (Hypertext Transfer Protocol) digunakan untuk mentransfer data di web. HTTPS adalah versi aman dari HTTP yang menggunakan enkripsi SSL/TLS. 5. Akses Basis Data: Banyak aplikasi web menggunakan basis data untuk menyimpan damengelola data. MySQL, PostgreSQL, dan MongoDB adalah beberapa contoh sistem basis data. 6. Server dan Hosting: Server web mengirimkan halaman web kepada pengguna melalui protokol HTTP. Hosting menyediakan tempat untuk menyimpan file dan data situs web agar dapat diakses secara online. 7. Framework dan Library: Framework dan library seperti React, Angular, dan Vue.js (untuk front-end) serta Django, Ruby on Rails, dan Express.js (untuk back-end) membantu mempercepat pengembangan dengan menyediakan struktur dan fungsionalitas yang telah dibuat sebelumnya. 8. Sistem Kendali Versi: Git adalah sistem kendali versi yang memungkinkan pengembang melacak perubahan dan mengedit kode program.
